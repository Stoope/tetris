# Tetris

Tetris game using webassembly (rust)

## Prerequisite

- [yarn](https://yarnpkg.com/en/docs/install)
- [wasm-pack](https://github.com/rustwasm/wasm-pack/blob/master/docs/src/setup.md#installing-wasm-pack)

## Installation

```sh
yarn
```

## Run

```sh
yarn start
```

## Build

```sh
yarn build
```
